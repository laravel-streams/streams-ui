## Introduction

Dropdowns are a sub-component that can be used to display dropdown content.

```json
{
    // ...
    "dropdown": {
        "home": {
            "text": "View Front",
            "attributes.href": "/",
            "attributes.target": "_blank"
        },
        "logout": {
            "text": "Logout",
            "attributes.href": "/logout"
        }
    }
}
```
